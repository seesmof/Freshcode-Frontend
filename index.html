Important Notes

    06.04.2023

  • Elements in a head section are the ones that don't actually appear on the page, but they are still important
  • Elements in a body section are the ones that our page will actually render
  • Live Server is a useful extension when we have a client-server interaction, meaning when we have some JavaScript on our site. However, it lags pretty oftenly, so I use this extension, called Live Preview, much better and opens right in the VS Code.
  • In HTML there are also headings, exactly six of them: h1, h2, h3, h4, h5, h6
  • Headings are useful not only for the looks, but also for a browser to identify the content of our website. That kind of touches on the semantic concept that we touched on briefly and will explore in more detail later. It all is for the meaning and CSS is for the looks.
  • When we work with an instrument everyday, we unintentionally learn it. Exact same thing with any programming language or whatever. We don't need to learn each tag there is, we just wanna get familiar with the general structure.
  • HTML Periodic Table is a useful table with all the most useful tags and their info.
  • A bit more about tags, the most important ones we will memorise pretty quickly
  • In tags there are also additional properties. Those are called attributes. It is a certain information about a particular element.
  • title attribute is a pop-up on hover that outputs some text.
  • Attributes can be global, like title, class, etc.
  • There are also boolean attributes. Those are the ones like disabled, they don't need to have a value, so you just add them and call it a day.
  • Some of the more popular boolean attributes
  • We somehow jumped right onto the forms and input tag, but okay :D
  • Input tag is a tag that is a prominent part of all forms
  • checked attribute works only on checkboxes and radio buttons. Means that it is checked by default. Useful for first buttons in a form.
  • So, gotta look into forms already. Alright!
  • name attribute for radio buttons associate them with one input sort of.
  • In head element there can be meta tags.
  • One of the useful and even essential meta tags in a head is charset. There we specify our encoding, which is usually UTF-8 or unicode, which represents all or almost all the written symbols out there.
  • HTML tag can and should always take lang attribute.
  • Now we're looking at Emmet plugin, pretty cool one.
  • Check out aria attributes. Those are for accessibility purposes.
